# Set it up

## Codebase setup and configuration

-   In the terminal run the following:

    -   git clone _copy the address_
    -   npm install

-   Create a nodemon.json with the following:

```
"env": {
        "SECRET_STRING": "",
        "MONGODB_USER": "lordmarkosa",
        "MONGODB_PASS": "",
    }
```

## Running the code

-   run: npm run fast
-   build: npm run build
